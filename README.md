# Imu Worlds - Coming Soon Page

## Overview

This is a "Coming Soon" page for Imu Worlds, a new online store. The page informs visitors that the store is under construction and provides an option for them to leave their email address to receive updates about the launch and announcements.

## Project Structure

- `index.html`: The main HTML file for the coming soon page.
- `styles.css`: The CSS file containing styles for the page.
- `app.js`: JavaScript file for any interactive functionality (currently linked but not provided in this sample).
- `images/`: Directory containing images used in the project, including the logo and favicon.
